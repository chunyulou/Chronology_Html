# 正覺大事記 - 使用說明

## 📁 檔案說明
- `event-table.html` - 主網頁檔案
- `table_data.js` - 資料檔案（從 Word 檔案提取的 210 筆資料）

## 🚀 使用方式
1. 確保兩個檔案在同一目錄下
2. 用瀏覽器開啟 `event-table.html`

## ✨ 功能特色

### 1. 分類 Tab（共 10 個）
- **總表**：顯示全部 210 筆資料，可垂直滾動
- **平實導師**：76 筆資料（從總表篩選）
- **講經**：32 筆資料（從總表篩選）
- **出版品**：96 筆資料（從總表篩選）
- **開課**：9 筆資料（從總表篩選）
- **禪修**：2 筆資料（從總表篩選）
- **視頻弘法**：18 筆資料（從總表篩選）
- **法會**：25 筆資料（從總表篩選）
- **宣導活動**：10 筆資料（從總表篩選）
- **祖師堂與正覺寺**：5 筆資料（從總表篩選）

### 2. 表格功能
- 三欄顯示：序號、日期、事件
- 總表顯示全部 210 筆資料
- 表格內容可垂直滾動
- 表頭固定，滾動時不會消失
- Tab 導覽列可水平滾動（不會垂直捲動）

#### 排序功能
- 點擊「日期」欄位標題可切換排序順序
- ▼ 向下三角形 = 升冪（時間由舊到新）- 預設
- ▲ 向上三角形 = 降冪（時間由新到舊）
- 排序會套用到當前顯示的分類

#### 搜尋功能
- 點擊「事件」欄位旁的 🔍 圖示開啟搜尋框
- 輸入關鍵字即時篩選資料（無需按Enter）
- 同時搜尋「事件」和「備註」欄位內容
- 符合的關鍵字會以螢光黃色標記
- 再次點擊 🔍 關閉搜尋並清除篩選

### 3. 備註功能
- 共 117 筆資料含有備註
- 有備註的事件旁會顯示 📝 圖示
- 點擊圖示開啟全螢幕 Modal
- Modal 內容可垂直滾動
- 點擊外部或按 ESC 鍵關閉

### 4. 響應式設計
- 支援桌面、平板、手機等各種螢幕尺寸
- Tab 可左右滑動（在小螢幕上）

## 🎨 設計風格
- 棕色系漸層主題
- 流暢的動畫效果
- 清爽的視覺設計

## 📊 資料來源
所有資料來自上傳的 Word 檔案：
`14-T002-備註版_20251027__韓劍華_1028_-清爽版__1_.docx`

## 🔧 技術說明
- 純 HTML + CSS + JavaScript
- 無需伺服器，可直接開啟
- 資料與邏輯分離，方便維護
- 自動根據關鍵字分類資料

## 💡 注意事項
1. 請勿分離兩個檔案，否則資料無法載入
2. 建議使用 Chrome、Firefox、Edge 等現代瀏覽器
3. 若要更新資料，請修改 `table_data.js` 檔案
