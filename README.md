# 正覺大事記 - 使用說明

## 📁 檔案說明
- `event-table.html` - 主網頁檔案
- `table_data.js` - 資料檔案（從 Word 檔案提取的 210 筆資料）

## 🚀 使用方式
1. 確保兩個檔案在同一目錄下
2. 用瀏覽器開啟 `event-table.html`

## ✨ 功能特色

### 1. 分類 Tab（共 10 個）
- **總表**：顯示前 20 筆資料，可垂直滾動
- **平實導師**：76 筆資料
- **講經**：32 筆資料
- **出版品**：96 筆資料
- **開課**：9 筆資料
- **禪修**：2 筆資料
- **視頻弘法**：18 筆資料
- **法會**：25 筆資料
- **宣導活動**：10 筆資料
- **祖師堂與正覺寺**：5 筆資料

### 2. 表格功能
- 三欄顯示：序號、日期、事件
- 總表最多顯示 20 筆，超過部分可垂直滾動
- 表頭固定，滾動時不會消失

### 3. 備註功能
- 共 117 筆資料含有備註
- 有備註的事件旁會顯示 📝 圖示
- 點擊圖示開啟全螢幕 Modal
- Modal 內容可垂直滾動
- 點擊外部或按 ESC 鍵關閉

### 4. 響應式設計
- 支援桌面、平板、手機等各種螢幕尺寸
- Tab 可左右滑動（在小螢幕上）

## 🎨 設計風格
- 棕色系漸層主題
- 流暢的動畫效果
- 清爽的視覺設計

## 📊 資料來源
所有資料來自上傳的 Word 檔案：
`14-T002-備註版_20251027__韓劍華_1028_-清爽版__1_.docx`

## 🔧 技術說明
- 純 HTML + CSS + JavaScript
- 無需伺服器，可直接開啟
- 資料與邏輯分離，方便維護
- 自動根據關鍵字分類資料

## 💡 注意事項
1. 請勿分離兩個檔案，否則資料無法載入
2. 建議使用 Chrome、Firefox、Edge 等現代瀏覽器
3. 若要更新資料，請修改 `table_data.js` 檔案
